# 🌍 World Layoffs Data Cleaning (SQL)

## 📌 Project Overview
This project focuses on cleaning a real-world layoffs dataset using SQL. The goal is to prepare the dataset for analysis by removing duplicates, handling missing values, standardizing text, and improving data consistency.

---

## 📂 Dataset
- Raw dataset: `layoffs.csv`
- Cleaned dataset: `cleaned_layoffs.csv`

---

## 🛠️ Data Cleaning Process
- Removed duplicate records
- Standardized text values
- Converted data types
- Handled NULL values
- Removed unnecessary rows
- Improved data consistency

---

## 📁 Project Files
- `layoffs.csv`
- `cleaned_layoffs.csv`
- `world_layoffs_data_cleaning.sql`
